Week 1: Node.js Basics
	•	Install Node.js, setup environment.
	•	Learn npm, modules, ES6 imports/exports.
	•	Async programming: callbacks, promises, async/await.
	•	Mini task: build a simple CLI app (e.g., TODO app in Node.js).
 
Week 2: Express.js Fundamentals
	•	Learn routing, middleware, error handling.
	•	CRUD APIs with in-memory storage.
	•	Postman basics for API testing.
	•	Deliverable: User CRUD API.
 
Week 3: Database Integration
	•	Learn MongoDB/PostgreSQL basics.
	•	Connect DB with Express.
	•	Implement REST API with DB.
	•	Deliverable: Notes app API with DB.
 
Week 4: Advanced Concepts
	•	JWT authentication, environment configs.
	•	Folder structure & best practices.
	•	Deliverable: Small backend service (Auth + CRUD) with Express.
